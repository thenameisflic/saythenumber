Say the Number API

This service converts numbers into their English word equivalents.

How to use:

1. API Endpoint

Make GET or POST requests to:

/num_to_english?number=YOUR_NUMBER

Examples:

/num_to_english?number=123
/num_to_english?number=-45.67

2. Web Interface

For a more user-friendly experience, visit our web interface:

/test

Rate Limiting

API requests are limited to 10 requests per minute per IP address.
